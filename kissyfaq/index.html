DOM与Node的区别

DOM demo

Node demo

通过上面两个demo，想必你已经大概明白了DOM和Node的区别了吧，DOM获取操作的都是dom对象，node 包括 dom , event , anim 模块的所有功能, 推荐采用 Node 模块, 你只需要把 KISSY.all 看做 jquery 中的 $ 就可以了。

建议一致使用 var $=KISSY.all; 操作dom.

如何引用KISSY组件

以drag&drop为例，只用use对应的模块名即可

打开demo，查看控制台->network，可以观察到，对应的js文件会被自动加载（可以一次加载多个模块，用逗号分割即可），是不是很方便呢？

在业务模块中可以以require的形式引用模块，查看demo
