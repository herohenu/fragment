KISSY 是什么?

KISSY是由阿里集团前端工程师们发起创建的一个开源 JS 框架。它具备模块化、高扩展性、组件齐全，接口一致、自主开发、适合多种应用场景等特性。

为什么要选择KISSY？

淘宝所有的业务都在使用，KISSY几乎能引用到任何场景，你还担心什么呢？ 除此之外，我们还提供全方位的支持：）

哪里可以下载KISSY？

KISSY所有代码都托管在GitHub上，包括最新的dev版，你都可以及时查看到。

KISSY支持哪些浏览器

淘宝几乎所有的页面都是基于KISSY的，所以KISSY会保证在所有主流浏览器中正常运行。主要包括以下浏览器：

*.js与*-min.js有什么区别

*-min.js是对应js的压缩版本，开发阶段引用*.js方便调试，发布到线上引用*-min.js大大减少文件体积。

DOM与Node的区别

DOM demo

Node demo

通过上面两个demo，想必你已经大概明白了DOM和Node的区别了吧，DOM获取操作的都是dom对象，node 包括 dom , event , anim 模块的所有功能, 推荐采用 Node 模块, 你只需要把 KISSY.all 看做 jquery 中的 $ 就可以了。

建议一致使用 var $=KISSY.all; 操作dom.
